Kapitel 1.4: JSON Daten generieren


Um die Datenbank mit JSON Daten zu befüllen, müssen JSON Daten beschafft werden. Dies könnte man mit existierenden JSON Datensätzen machen, wie z.B. in diesem GitHub zu finden sind. Allerdings sind diese Datensätze sehr groß und am Anfang nicht lehrfreundlich, daher werden in dieser Lehrlektion sämtliche JSON Datensätze mithilfe von Python Code und dem Faker Modul generiert.



Die JSON Datenstruktur sowie erlaubte Typen lassen sich unter JSON.org finden. Hier finden sie Generator, die die meisten Kombinationen an JSON Datentypen und Strukturen generieren kann.
