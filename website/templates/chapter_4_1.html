Kapitel 4.1: JSON Objekte ausgeben


Objekte Ausgeben

Um JSON Daten aus der Datenbank als JSON Objekte auszugeben, gibt es den JSON_OBJECT Befehl. Dieser nimmt eine beliebige Menge an KEY-VALUE Paaren, und fügt diese zu einem JSON Objekt zusammen. Der Befehl lässt sich verschachteln, um verschachtelte JSON Objekte zu generieren.

Um mehrere JSON Objekte in ein JSON Objekt zu kombinieren, gibt es in MariaDB und MySQL den Befehl JSONOBJECTAGG, in SQLITE heißt der equivalente Befehl JSON_GROUP_OBJECT.

JSON_OBJECT

JSON_Object Syntax

JSON_OBJECT([KEY, VALUE], ...)

JSON_OBJECT Syntax Beispiel

JSON_OBJECT(Vorname, Kevin, Nachname, Müller)

JSON Ergebnis
                        {
                            "Vorname": "Kevin",
                            "Nachname": "Müller
                        }
JSON_OBJECT Verschachtelung Syntax Beispiel

JSON_OBJECT(Vorname, Kevin, Nachname, Müller, Urlaubsziele, JSON_OBJECT(Ziel_1, Australian, Ziel_2, Neuseeland))

JSON Ergebnis
                        {
                            "Vorname": "Kevin",
                            "Nachname": "Müller,
                            "Urlaubsziele": {
                                "Ziel_1": "Australian",
                                "Ziel_2": "Neuseeland"
                                }
                        }
