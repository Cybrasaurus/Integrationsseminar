Kapitel 1.2: Datenbank erstellen


SQL Datenbanken erstellen

Um mit praktischen Beispielen arbeiten zu können, müssen zuerst Datenbanken für die jeweiligen Datenbank-Systeme erstellt werden. Dies kann mit den Buttons auf der rechten Seite gemacht werden. Hier wird zugleich ein "Cursor" erstellt, welcher als Console für die jeweilige Datenbank dient, sodass normale SQL Commands verwendet werden können.
