Kapitel 3.1: Abfrage zerteilter JSON Objekte


Daten abfragen

Um aufgeteilte JSON Objekte abzufragen, ändern oder zu löschen, müssen diese als SQL-Datentypen abgespeichert werden, wie in Kapitel 1.5 unter dem Punkt "JSON als SQL Datentypen" demonstriert.

Dies läuft über normale SQL Queries, die sich zwischen MySQL, SQLITE und MARIADB kaum unterscheiden. Da der Schwerpunkt dieser Lernlektion nicht SQL im allgemeinen, sondern die Verarbeitung von JSON in relationalen Datenbanken ist, wird an dieser Stelle nicht weiter auf die Details von Query Statements eingegangen

Abfragen

SELECT [COLUMN_NAMES] FROM [TABLE_NAME] WHERE [CONDITION]

Ändern

UPDATE [TABLE_NAME]
SET [COLUMN_NAME] = [VALUE], ...
WHERE [CONDITION]

Löschen

DELETE FROM [TABLE_NAME] WHERE [CONDITION]
